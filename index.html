Calculadora Fibonacci em Java

Este código imprime os primeiros N números da sequência de Fibonacci.

public class Fibonacci { public static void main(String[] args) { int n = 10, a = 0, b = 1; System.out.print("Fibonacci: "); for (int i = 0; i < n; i++) { System.out.print(a + " "); int temp = a; a = b; b = temp + b; } } }

Contador de Dias para o Aniversário

Este código em Python calcula quantos dias faltam (ou passaram) para o seu aniversário neste ano.

from datetime import date # pega data de hoje hoje = date.today() # pede o aniversário aniversario = input("Qual seu aniversário? (dd/mm): ") dia, mes = map(int, aniversario.split('/')) # monta a data do aniversário deste ano aniversario_ano = date(hoje.year, mes, dia) # calcula a diferença if hoje > aniversario_ano: print("Já passaram", (hoje - aniversario_ano).days, "dias do seu aniversário.") elif hoje == aniversario_ano: print("Parabéns! Hoje é seu aniversário!") else: print("Seu aniversário é daqui", (aniversario_ano - hoje).days, "dias.")
